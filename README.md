# laser-design

腾讯地图原生绘制工具库 - 提供丰富的地图绘制和标注功能

## 📦 安装

```bash
npm install laser-design
```

### 基础图标绘制

```typescript
import { BaseIconDrawer } from 'laser-design';

const icon = new BaseIconDrawer({
  map: map,
  lat: 30.244831,
  lng: 120.151634,
  type: 'UAV-Compliant',
  width: 40,
  height: 40
});
```
